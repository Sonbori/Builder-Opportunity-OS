# Builder Opportunity OS for Codex

Read `CLAUDE.md` first. It defines the operating model for this repository.

Key rules:

- This repo is an opportunity operating system, not only a job-search tool.
- Reuse the existing modes, scripts, templates, and tracker flow where possible.
- Keep user-specific positioning in `config/profile.yml`, `modes/_profile.md`,
  `article-digest.md`, and `portals.yml`.
- Never submit applications, proposals, or messages without review.
- Prefer selective, high-signal opportunities over volume.
